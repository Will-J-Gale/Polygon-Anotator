## Shortcuts

| Key           | Action        |
| ------------- |:-------------:|
| Ctrl + S      | Save          |
| Ctrl + C      | Copy Polygon  |
| Ctrl + V      | Paste Polygon |
| Del           | Delete Polygon|
| W             | Free draw polygon|
| D             | Next Image    |
| A             | Previous Image|
| Mouse Wheel   | Zoom +/-      |
| Mouse Wheel drag |Scroll around |
| 0-9           | Change color|
